HTML Tags
< b > Makes the text bold This is the paragraph that are in bold.
< i > Makes the font style Italic This is the paragraph that are written in italic.
< sub > Defines subscript text This is a subscript text H2O.
< sup > Defines superscript text This is an equation (x2 + y2).
< u > Used to underline the text The text is underlined.
< mark > Used to mark the text This text is marked.
< a > Defines a hyperlink, which is used to link from one page to another To Learn More about anchor tag Click Here
< img > Used to embed an image in an HTML page
< video > Used to embed video content in a HTML document
< audio > Used to embed audio content in a HTML document
< iframe > An inline frame is used to embed another document within the current HTML document
< br > It inserts a single line break
